# DESIGN.md

## 气质与意象
阳光午后，一个华裔小朋友坐在明亮的书房里，打开平板准备中文识字闯关。屏幕上是一个温暖的卡通小猴子导游，每完成一关就跳起来鼓掌。色彩像水果糖一样鲜明但不刺眼，圆角大按钮让小手轻松点击。

## 视觉策略
- 图像方向：扁平卡通插画，线条圆润、无阴影
- 图形语言：大量圆角卡片、气泡对话框、星星/徽章图标
- 插画角色：一只穿红色小马甲的卡通小猴子作为系统导游

## 配色方案
| 角色 | 色值 | 意象来源 |
|------|------|---------|
| 主色（Primary） | #FF6B35 | 暖橘色——猴子小马甲、按钮高亮 |
| 辅助色（Secondary） | #4ECDC4 | 清新薄荷绿——进度条、成功状态 |
| 强调色（Accent） | #FFE66D | 阳光黄——星星、徽章、分数高亮 |
| 背景主色 | #FFF8F0 | 暖白奶油——温暖不刺眼 |
| 卡片背景 | #FFFFFF | 纯白卡片 |
| 文字主色 | #2D3436 | 深炭灰——高对比度易读 |
| 文字辅助 | #636E72 | 中灰——次要信息 |
| 错误/警告 | #FF6B6B | 柔和珊瑚红 |
| 成功 | #51CF66 | 清新绿 |

## 字体排版
- 标题字体：ZCOOL KuaiLe（Google Fonts .cn），卡通手写风
- 正文字体：Noto Sans SC（Google Fonts .cn），清晰易读
- 数字字体：ZCOOL KuaiLe（与标题一致，保持游戏感）
- 排版节奏：标题大而醒目（32-48px），正文舒适（16-18px），按钮文字粗大（18-20px bold）

## 动效与交互
- 运动性格：弹性、欢快、鼓励性
- 缓动曲线：ease-out-back（弹出感）
- 悬停：按钮 scale(1.05) + 微微上浮阴影
- 过渡：页面切换 fade + slide，300ms
- 答对反馈：星星飞入 + 经验值数字弹出上浮消失
- 答错反馈：温柔摇头动画，无惩罚感
- 倒计时：最后30秒数字变红并脉动

## 页面结构
- 首页：大标题 + 小猴子插画 + "开始测试"大按钮，居中布局
- 用户信息页：卡片式表单，圆角大输入框，国家/语言用图标+下拉
- 测试页：顶部倒计时条 + 进度星星，中间题目区，底部选项按钮
- 结果页：大数字展示 + 星级评分 + 徽章墙
- 管理后台：简洁数据表格，与前台风格区分（偏专业工具风格）

## 设计禁忌
- 禁止使用考试/卷子的视觉隐喻（不要红色批改、不要分数排名）
- 禁止冷色调科技蓝/紫渐变
- 禁止小字体密排的表格用于儿童界面
- 禁止刺眼纯红用于错误提示
- 禁止弹窗确认中断游戏流程
